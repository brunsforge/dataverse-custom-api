## Auth-Konfiguration

Für lokale Tests liegen Beispielkonfigurationen im Projektroot:

- auth.example.json
- auth.clientsecret.example.json
- auth.devicecode.example.json
- auth.interactivebrowser.example.json

Kopiere die passende Datei und benenne sie in `auth.json` um oder kopiere sie per Script an diese Stelle.
